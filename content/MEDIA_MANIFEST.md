## Usage

### Option A: Direct URLs (Recommended for dev)
Use these URLs directly in your code. No hosting costs.

### Option B: Self-hosted
1. Download all images using the URLs above
2. Upload to S3/Cloudflare R2/your CDN
3. Update image paths in code

### Download Script
```bash
# Download all images from manifest
grep -E '^https://' MEDIA_MANIFEST.md | while read url; do
  wget -q "$url" -P ./images/
done
```
